# 研究室紹介ウェブサイト

このリポジトリは、研究室を紹介するためのウェブサイトのソースコードです。

## 🎯 ウェブサイトの構成

### ページ一覧

| ページ | ファイル名 | 説明 |
|--------|----------|------|
| ホーム | `index.html` | トップページ。研究室への入口 |
| 研究室紹介 | `about.html` | 研究室の概要や特徴 |
| 主な研究 | `research.html` | 現在進行中の研究テーマ一覧 |
| 研究論文 | `papers.html` | 発表した学術論文一覧 |
| 学位論文 | `thesis.html` | 卒業論文・修士論文・博士論文一覧 |
| 統計 | `statistics.html` | 研究実績の統計情報 |

## 📁 ディレクトリ構造

```
lab-website/
├── index.html                      # トップページ
├── about.html                      # 研究室紹介
├── research.html                   # 主な研究
├── papers.html                     # 研究論文リスト
├── thesis.html                     # 卒論・修論・博論
├── statistics.html                 # 研究内容の統計
├── css/
│   └── style.css                   # ウェブサイトのスタイル
├── js/
│   └── script.js                   # JavaScriptの基本機能
├── images/                         # 画像ファイルディレクトリ
└── README.md                       # このファイル
```

## 🎨 デザイン特徴

- **レスポンシブデザイン**: モバイル、タブレット、デスクトップに対応
- **モダンデザイン**: グラデーション背景とフラットデザインを採用
- **使いやすいナビゲーション**: ページ間の移動が簡単
- **プロフェッショナルな見た目**: 学術的なウェブサイトにふさわしいデザイン

## 🚀 技術スタック

- **HTML5**: ウェブページの構造
- **CSS3**: スタイリングとレスポンシブデザイン
- **JavaScript**: インタラクティブな機能

## 💡 カスタマイズ方法

### コンテンツの変更

各HTMLファイルを編集して、以下の情報を更新してください：

1. **研究室の名前と説明**: `about.html` の指導教員情報などを更新
2. **研究テーマ**: `research.html` に研究テーマを追加
3. **論文リスト**: `papers.html` で発表した論文を記載
4. **学位論文**: `thesis.html` で取得者の論文情報を記載
5. **統計情報**: `statistics.html` で実績を数字で記載

### スタイルのカスタマイズ

`css/style.css` の色やレイアウトを変更することで、ウェブサイトの見た目を調整できます。

主な色設定：
- プライマリカラー: `#3498db` (青)
- ダークカラー: `#2c3e50` (濃紺)

### 画像の追加

画像を使用する場合は、`images/` ディレクトリに配置し、HTMLで以下のように参照します：

```html
<img src="images/filename.jpg" alt="説明">
```

## 📝 注意事項

- 各HTMLファイルは日本語に対応しています
- メタ文字セットに `UTF-8` を使用しています
- ナビゲーションバーは各ページで自動的に更新されます
